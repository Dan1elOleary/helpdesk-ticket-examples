# Help Desk Ticket Template

## Ticket Title

Briefly describe the issue in one clear sentence.

**Example:**  
User unable to access Outlook due to profile loading error

---

## Ticket Information

| Field | Details |
|---|---|
| User | Example User |
| Department | Example Department |
| Device Name | DESKTOP-EXAMPLE |
| Operating System | Windows 11 |
| Application/System | Microsoft Outlook / Microsoft 365 |
| Priority | Low / Medium / High |
| Status | Open / In Progress / Resolved / Closed |

---

## Issue Summary

Describe the issue reported by the user.

**Example:**  
The user reported that Microsoft Outlook would not open correctly and displayed an error related to the Outlook profile. The issue prevented the user from accessing email through the desktop application.

---

## User Impact

Explain how the issue affected the user or business workflow.

**Example:**  
The user was unable to send or receive email through the Outlook desktop app. Webmail access through Outlook on the web was still available.

---

## Environment

Document relevant system, application, or account information.

- Device: Windows 11 workstation
- Application: Microsoft Outlook
- Account Type: Microsoft 365
- Network: Corporate network / VPN / Remote connection
- User Access Level: Standard user

---

## Troubleshooting Steps

Document each step taken during troubleshooting.

1. Confirmed the issue with the user.
2. Verified whether the issue occurred in Outlook desktop only or also in Outlook on the web.
3. Checked internet connectivity.
4. Restarted the Outlook application.
5. Restarted the workstation.
6. Checked for Microsoft 365 service issues.
7. Reviewed account sign-in status.
8. Performed additional troubleshooting as needed.

---

## Findings

Summarize what was discovered during troubleshooting.

**Example:**  
Outlook on the web worked correctly, which confirmed that the mailbox and Microsoft 365 account were functional. The issue appeared to be isolated to the local Outlook desktop profile.

---

## Root Cause

Document the likely or confirmed cause of the issue.

**Example:**  
The local Outlook profile appeared to be corrupted, preventing Outlook from loading the mailbox correctly.

---

## Resolution

Explain what fixed the issue.

**Example:**  
Created a new Outlook profile through Control Panel Mail settings and set it as the default profile. Outlook launched successfully and began syncing the mailbox.

---

## User Communication

Document what was communicated to the user.

**Example:**  
Informed the user that the Outlook desktop profile was rebuilt and that email may take some time to fully resync. Confirmed that Outlook was opening successfully before closing the ticket.

---

## Follow-Up Notes

Add any follow-up actions or monitoring notes.

**Example:**  
Advised the user to report any missing mail, calendar issues, or repeated Outlook errors. No further issues were reported after testing.

---

## Closure Summary

Provide a concise final ticket note.

**Example:**  
Resolved issue by rebuilding the local Outlook profile. Confirmed successful Outlook launch and mailbox sync with the user. Ticket closed.
